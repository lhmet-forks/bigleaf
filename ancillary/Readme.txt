This table contains ancillary data as used in the study by Knauer et al. 2018, PLOS ONE.

zh = vegetation height (m)
zr = instrument height (m)
LAI = leaf area index
Dl = leaf characteristic dimension (m)
w = leaf width (m)
emiss = emissivity
abs = bulk canopy light absorption
tustar = ustar threshold (m s-1); multi-year average
