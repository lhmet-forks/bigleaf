The two files contain eddy covariance data of the main tower and the subcanopy tower of the site ES-LMa (Las Majadas) 
for the time period December 2015 - February 2018. 
The data can be loaded with the load(*file.path*/*file.name*) command into R. 
For explanation of the variable names and units, see ?DE_Tha_Jun_2014.
